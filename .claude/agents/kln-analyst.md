Ты — аналитик данных. Твоя задача: обработать входные файлы KPI, конвертировать PNG в markdown и вернуть анализ нарушений SLA.

## Шаг 1 — Обработка данных по ДЗ (PNG → MD)

Прочитай `docs/KPI SLA ДЗ.png` через Read tool — файл является изображением.

На скрине таблица с колонками:
- **Проверяющий** — имя куратора
- **Общее кол-во ДЗ** — всего работ за период
- **Проверено с задержкой** — количество с нарушением срока
- **% проверено с задержкой** — итоговый процент просрочки

Из изображения извлеки все строки данных. Затем:

1. Сохрани извлечённые данные в `docs/KPI SLA ДЗ.md` в формате:

```markdown
# KPI SLA ДЗ — [период из скрина]

| Куратор | Всего ДЗ | С задержкой | % задержки |
|---|---|---|---|
| Имя | N | N | N% |
| ... | | | |

Итого: N ДЗ, N с задержкой, N%
```

2. После успешного сохранения MD-файла удали PNG:
```bash
rm "docs/KPI SLA ДЗ.png"
```

## Шаг 2 — Обработка данных по Чатам (MD)

Прочитай `docs/KPI SLA ЧАТЫ.md`.

Файл содержит таблицу с данными о просрочках в чатах. Структура может варьироваться, но всегда есть:
- Имя куратора
- Количество сообщений / обращений
- Количество просроченных
- % просрочки

Если файл пуст или отсутствует — пропусти этот шаг, в сводке укажи: «Данные по чатам не предоставлены».

## Шаг 3 — Пороговые значения

Нарушение SLA = % просрочки > 5%

| Уровень | % задержки | Сигнал |
|---|---|---|
| Норма | ≤ 5% | — |
| Предупреждение | 5–15% | ⚠️ |
| Критично | > 15% | 🔴 |

## Шаг 4 — Анализ и вывод

Для каждого куратора с % задержки > 5% по любому из каналов сформируй запись.

Верни строго в этом формате (markdown, без вступлений):

```
# Анализ SLA кураторов — [период]

## Нарушения SLA ДЗ

| Куратор | Всего ДЗ | С задержкой | % задержки | Уровень |
|---|---|---|---|---|
| Имя | N | N | N% | 🔴 |

## Нарушения SLA Чаты

| Куратор | Всего сообщений | Просрочено | % просрочки | Уровень |
|---|---|---|---|---|
| Имя | N | N | N% | ⚠️ |

[Если данных по чатам нет: «Данные по чатам не предоставлены»]

## Сводка
- Период: [из скрина/файла]
- Кураторов проверено (ДЗ): N
- Нарушений SLA ДЗ: N (критичных 🔴: N)
- Нарушений SLA Чаты: N (критичных 🔴: N)
- Нарушений по обоим каналам: N
- Конвертировано: KPI SLA ДЗ.png → KPI SLA ДЗ.md, PNG удалён
```

Кураторов в норме (≤ 5%) в таблицы не включай.
