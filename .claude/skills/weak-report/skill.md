## Пайплайн: Еженедельный отчёт "Слабые показатели"

Запускай субагентов **последовательно**:

### Шаг 1 — Анализ данных (kln-analyst)

Запусти субагента `kln-analyst`:

```
Используй агента kln-analyst. Прочитай последний файл КЛН в папке data/ и верни структурированный анализ слабых показателей по всем кураторам.
```

Дождись полного ответа — это входные данные для следующего шага.

### Шаг 2 — Форматирование отчёта (report-formatter)

Передай результат шага 1 субагенту `report-formatter`:

```
Используй агента report-formatter. Вот результат анализа КЛН:

[ВСТАВЬ ПОЛНЫЙ ВЫВОД kln-analyst]

Сформируй финальный отчёт и сохрани в папку reports/.
```

### Шаг 3 — Подтверждение

После завершения обоих шагов выведи:
- Путь к созданному файлу отчёта
- Краткую сводку: сколько кураторов со слабыми показателями из общего числа

---

**Тип пайплайна:** последовательный (результат агента 1 → вход агента 2)  
**Входные данные:** файлы КЛН в папке `data/`  
**Выходной артефакт:** отчёт в `reports/ГГГГ-ММ-ДД-weak-curators-report.md`
