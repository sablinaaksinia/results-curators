name: start-potok
description: Анализирует файл data/запуски-потоков.md и составляет сообщение для Telegram-чата с потоками, стартующими на следующей неделе
---

Прочитай файл `data/запуски-потоков.md`.

Определи диапазон следующей недели: с ближайшего понедельника по воскресенье включительно (относительно сегодняшней даты).

Найди все потоки, у которых **Дата старта** попадает в этот диапазон.

Составь сообщение для Telegram строго в следующем формате:

```
🚀 ЗАПУСКИ ПОТОКОВ

ДД.ММ — Название потока — Куратор
ДД.ММ — Название потока — Куратор
...
```

Правила:
- Строки отсортированы по дате старта (от меньшей к большей)
- Если в один день несколько потоков — каждый на отдельной строке
- Если куратор не указан — написать «куратор не назначен»
- Если потоков на следующей неделе нет — написать: «На следующей неделе запусков нет»
- Никакого лишнего текста до и после сообщения — только готовый текст для вставки в чат
